QLRanks quick ELO overview
=====================

A console application for fetching the ELO of a player from [qlranks.com](http://www.qlranks.com)
and periodically updating it.

## Requirements

* Node

## How to run

Install dependencies using:

``` bash
$ npm install
```

Run using:

``` bash
$ npm install
```

Run tests using:

``` bash
$ npm test
```
